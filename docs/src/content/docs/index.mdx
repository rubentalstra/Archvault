## Features

<CardGrid stagger>
  <Card title="C4 Modeling" icon="pencil">
    Model your software architecture using the C4 model — System Context (L1), Container (L2), and Component (L3) diagrams.
  </Card>
  <Card title="Visual Editor" icon="setting">
    Drag-and-drop diagram editor powered by React Flow with auto-layout, connection routing, and real-time autosave.
  </Card>
  <Card title="Organizations & Workspaces" icon="open-book">
    Organize architecture work into organizations and workspaces with team-based collaboration.
  </Card>
  <Card title="Auth & SSO" icon="approve-check-circle">
    Enterprise-ready authentication with email/password, two-factor auth, SSO (SAML/OIDC), and SCIM provisioning.
  </Card>
  <Card title="Role-Based Access Control" icon="warning">
    Fine-grained permissions with organization-level roles, member management, and team-based access.
  </Card>
  <Card title="Internationalization" icon="translate">
    Fully internationalized interface with English and Dutch out of the box, easily extensible to more locales.
  </Card>
</CardGrid>
