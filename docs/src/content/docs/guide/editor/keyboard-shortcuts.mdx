## General

| Shortcut | Action |
|----------|--------|
| <Kbd mac="Cmd+Z" windows="Ctrl+Z" linux="Ctrl+Z" /> | Undo |
| <Kbd mac="Cmd+Shift+Z" windows="Ctrl+Shift+Z" linux="Ctrl+Shift+Z" /> | Redo |
| <Kbd mac="Delete" windows="Delete" linux="Delete" /> | Remove selected |
| <Kbd mac="Cmd+A" windows="Ctrl+A" linux="Ctrl+A" /> | Select all |
| <Kbd mac="Cmd+C" windows="Ctrl+C" linux="Ctrl+C" /> | Copy |
| <Kbd mac="Cmd+V" windows="Ctrl+V" linux="Ctrl+V" /> | Paste |
