The fastest way to try ArchVault is with Docker Compose. This guide walks you through cloning the repo, configuring environment variables, and starting the application.

## Prerequisites

- Docker and Docker Compose installed
- Git

## Steps

1. Clone the repository
2. Copy the example environment file
3. Start the containers
4. Open the application in your browser
